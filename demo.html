标题 & 段落 CommonMark
Markdown 输入
# Heading 1 ## Heading 2 ### Heading 3 This is a paragraph with **bold** and *italic* text. Another paragraph with `inline code`.

<h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3> <p>This is a paragraph with <strong>bold</strong> and <em>italic</em> text.</p> <p>Another paragraph with <code>inline code</code>.</p>

语法高亮 增强功能

HTML 输出（带高亮）
<pre><code class="language-rust"> <span class="hl-keyword">fn</span> fibonacci(n: <span class="hl-type">u32</span>) -> <span class="hl-type">u32</span> { <span class="hl-keyword">match</span> n { 0 => <span class="hl-number">0</span>, 1 => <span class="hl-number">1</span>, <span class="hl-keyword">_</span> => fibonacci(n - <span class="hl-number">1</span>) + fibonacci(n - <span class="hl-number">2</span>), } } </code></pre>
预览效果（带高亮）
fn fibonacci(n: u32) -> u32 {
    match n {
        0 => 0,
        1 => 1,
        _ => fibonacci(n - 1)
             + fibonacci(n - 2),
    }
}
